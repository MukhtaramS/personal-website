# Mukhtaram Sulaimonov — Personal Portfolio

Personal portfolio website showcasing my experience, projects, and skills.

## Stack

- **React 19** — UI framework
- **Create React App** — project bootstrapping and build tooling
- **Plain CSS** — styling, no CSS framework
- **GitHub Pages / Vercel** — hosting

## Run locally

```
npm install
npm start
```

## Build

```
npm run build
```
